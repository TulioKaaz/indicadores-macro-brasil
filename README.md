# Indicadores Macro Brasil

Análise exploratória de indicadores macroeconômicos brasileiros (IPCA e câmbio USD/BRL), com dados extraídos diretamente da API do Banco Central (SGS).

## Objetivo
Projeto de estudo aplicado, parte da minha transição de carreira para Análise de Dados/Finanças. Serve de base para conteúdo publicado no LinkedIn.

## Como rodar
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Estrutura
- `notebooks/` — notebooks de análise
- `outputs/` — gráficos e resultados gerados

## Fonte de dados
- Banco Central do Brasil — Sistema Gerenciador de Séries Temporais (SGS)
